## 一、产品本质（⚠️ 非常重要）

> **面向电商客服 / 客服主管 / 客服经理的「工具订阅型产品」**

- 当前阶段：**所有工具免费**
- 未来阶段：**工具订阅制 / 高级功能收费**
- 「职业成长」只是 **工具中心中的一个子模块**
- 产品核心价值：**解决客服真实工作问题，而不是教课**

## 二、目标用户与真实使用场景

### 2.1 目标用户
- 电商客服（售前 / 售后）
- 客服组长
- 客服主管（10–30 人团队）
- 客服经理 / 服务运营

### 2.2 高频使用场景
- 日常客服工作提效
- 客诉 / 投诉处理
- KPI 计算与复盘
- 管理汇报与总结
- SOP / 话术 / 文档快速复用

---

## 三、基础产品约束（必须遵守）

### 3.1 登录与身份体系
- 默认 **匿名使用**（anonymous_id）
- 产品启动流程：
    1. Splash 页面
    2. 判断是否已绑定邮箱
        - 已绑定 → 直接进入首页
        - 未绑定 → 提示：
            - 绑定邮箱
            - 或跳过（继续匿名）
- 匿名数据在绑定邮箱后 **必须合并**

### 3.2 其他

1. pc和移动端都有设计稿，双端都需要高体验响应式，但是html固定font-size:16px，按这个要求先产出适配的规则约束；
2. 需要封装多种主题，var变量结合tailwindCSS设计系统，包括但不限于颜色、阴影、渐变、字体；
3. 目前需要三种主题：暗/亮/hero，默认hero主题，hero主题需要参考最新的英雄联盟客户端高贵主题；
4. 涉及阴影的动画、变换、显隐，需要保证不突兀；
